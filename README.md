 NayePankh Foundation Website

A responsive NGO website built using HTML and CSS for the fictional NayePankh Foundation. The website features a modern and clean design with sections for the organization's mission, focus areas, impact statistics, and contact information.

Features

- Responsive design for desktop, tablet, and mobile devices
- Modern user interface with attractive color gradients
- Hero section with call-to-action button
- Program showcase cards with images
- Impact statistics section
- Contact information section
- Smooth scrolling navigation


Technologies Used

- HTML5
- CSS3

Purpose

This project was created to demonstrate front-end web development skills by designing a professional NGO landing page with a responsive layout and modern styling.
